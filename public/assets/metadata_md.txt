# 🪇 Burnout Buddy – Asset Metadata Overview (v1.0)

This file explains what’s in the Burnout Buddy asset ZIP and how the
metadata files (`manifest.json` and `cards.json`) map to the image assets.

---

## 1. Folder Layout (expected)

All assets live under a single root folder, for example:

`burnout-buddy-assets-v1/`
- `cards/`
  - `main-fronts/`
  - `main-backs/`
  - `responses/`
- `icons/`
- `metadata/`
